# Space Invaders — Game Overview

A Python/Pygame implementation of the classic arcade shooter.

## Controls
- Left / Right arrow keys: move the ship
- Spacebar: fire bullets

## Files
- `space_invaders_game.py` — main game loop
- `alien.png`, `battleship.png`, `bgspace.jpg`, `blt2.png`, `explosion.wav`, `laser.wav`, `icon.png` — assets
